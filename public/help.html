process configuration

Processes are fetched with the ps command.
Here is how your process configuration should look like:

processMonitor:
    refresh: 1000
option example comment
refresh 1000

Time between each processes retrieval.

Do not set a too small interval, the retrieval + parsing of running processes is a costly operation.

mysql configuration

Running mysql processes are fetched using the SHOW PROCESSLIST query.
Disk usage is computed with the du command.
Here is how your mysql configuration should look like:

mysqlMonitor:
    connection:
        user:     root
        password: root
        host:     127.0.0.1
        port:     3306
    refresh: 1000
    datadir: /data/mysql
option example comment
connection.user root mysql user
connection.password root mysql user password
connection.host localhost mysql host
connection.port 3306 mysql port
refresh 1000

Time between each processes retrieval.

Do not set a too small interval, the retrieval + parsing of running processes is a costly operation.
datadir /data/mysql mysql datadir, used to get information about disk usage
